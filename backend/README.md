# grad-paddy backend

## Quick Start

### Prerequisites
- Python 3.10+ and uv installed.

### Install dependencies
```bash
uv sync
```

### Start the development server

```bash
uv run fastapi dev
```

Visit [http://localhost:8000](http://localhost:8000)

## Project Structure

- `main.py` - Your FastAPI application
- `pyproject.toml` - Project dependencies
